Tested on Ubuntu 18.04

1.) Install compiler:

sudo add-apt-repository ppa:ubuntu-toolchain-r/test

sudo apt update

sudo apt-get install gcc g++

sudo apt-get install gcc-9 g++-9

sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-9 900 --slave /usr/bin/g++ g++ /usr/bin/g++-9

sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-7 700 --slave /usr/bin/g++ g++ /usr/bin/g++-7

you can keep the standard compiler as default using the cmd: sudo update-alternatives -- config gcc

2.) Install Cmake: (>=3.14)

sudo snap install cmake

3.) Prerequistes

sudo apt-get install libgtk-3-dev pkg-config

sudo apt-get install curl unzip tar

*****TANGO_ONLY*************

sudo apt-get install libtango-dev

sudo apt-get install libtango9

install omniORB from sources http://omniorb.sourceforge.net/index.html

**ENDE*TANGO_ONLY*************

sudo ln -s /usr/bin/python3 /usr/bin/python

on linux boost >= 1.71 must be installed, usually by : sudo apt-get install libboost-all-dev

BUT: on Ubuntu 18 or lower boost must be installed from sources:

"https://www.boost.org/doc/libs/1_72_0/more/getting_started/unix-variants.html"

tar --bzip2 -xf /path/to/boost_1_72_0.tar.bz2

cd boost_1_72_0

./bootstrap.sh --prefix=/usr/

./b2

sudo ./b2 install

sudo apt-get install qt5-default

sudo apt-get install libqt5svg5-dev

sudo apt-get install libqt5webkit5-dev

sudo apt-get install qtwebengine5-dev

4.) install vcpkg (C++ package manager)

cd /

sudo mkdir vcpkg

sudo chmod 777 vcpkg

es empfehlen sich kurze Pfade , z.B. von / aus

git clone https://github.com/microsoft/vcpkg.git

cd vcpkg

sudo ./bootstrap-vcpkg.sh

./vcpkg integrate install

5.) Install magic-enum,libuv, zeromq with vcpkg

sudo ./vcpkg install magic-enum

sudo ./vcpkg install libuv

sudo ./vcpkg install zeromq
